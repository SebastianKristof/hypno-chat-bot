Welcome to the HypnoBot API

HypnoBot is an advanced hypnotherapy chatbot designed to provide relaxation and therapeutic conversation using natural language processing techniques. This API allows you to integrate HypnoBot into your applications.

API Endpoints

The following endpoints are available in the HypnoBot API:

GET /api/health

Check the health status of the API server.

Response: JSON with status information.

POST /api/chat

Send a message to the hypnotherapy bot.

Request Body: JSON with user input.

Response: JSON with bot's response.
